# Horiseon-Accessibility
Horiseon refactoring for accessibility 
In this exercise, I refactored the provided starter code from the Horiseon webpage to reflect updated accessibility standards for search engine optimization. In addition to the accessibility updates, I consolidated semantic structure and CSS selectors.

The link to the deployed application is:  https://sopeethong1.github.io/Horiseon-Accessibility/
